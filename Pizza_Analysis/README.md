# 🍕 Pizza Sales Analysis (SQL)

This project analyzes a pizza sales dataset to uncover key business insights using MySQL.

---

## 📌 Objectives

- 💰 Calculate total revenue, average order value, and pizza sales volume.
- 📅 Understand daily and monthly sales trends.
- 📊 Identify best and worst performing pizzas by revenue, quantity, and orders.
- 🧠 Perform category-based and size-based sales analysis.

---

## 🗂️ Key SQL Concepts Used

- Aggregation (`SUM`, `AVG`, `COUNT`)
- `GROUP BY` and `ORDER BY`
- `LIMIT` for ranking
- Date functions for day/month trends

---

## 🧪 Sample KPIs

- **Total Revenue**  
- **Average Order Value**  
- **Top 5 & Bottom 5 Pizzas** by:
  - Revenue
  - Quantity
  - Number of Orders  
- **Sales by Pizza Category & Size**
